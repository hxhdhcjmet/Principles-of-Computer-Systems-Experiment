\subsection{实验目的}
\begin{itemize}
    \item 学习矩阵乘法程序优化:在考虑数据大小,线程数,硬件配置,程序算法等的前提下考虑矩阵乘法程序的优化. 
    \item 理解Cache机制:通过改变访存方式,验证时间局部性与空间局部性对程序执行效率的影响.进而最终理解Cache对程序执行效率的巨大影响.
    \item 探索并行计算机制:利用多线程与指令级并行(SIMD/循环展开),分析硬件资源并行优化带来的加速比.
    \item 掌握跨平台工程能力:学习交叉编译和QEMU模拟器的使用,学习掌握RISC-V工具链的基本使用方法.学习RISC-V Linux内核编译并通过QEMU运行.
\end{itemize}
